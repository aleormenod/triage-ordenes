# Triage Ordenes - Generador de Órdenes Preoperatorias

Sistema para generar órdenes automáticas con branding personalizado del doctor.

## Features
- Pre-Orden de Presupuesto
- Orden de Exámenes Preoperatorios
- Orden de Electrocardiograma
- Personalización con branding
- Descarga PDF

vercel
