# 🚀 DEPLOY A VERCEL — TRIAGE ORDENES

Tu código está en GitHub: https://github.com/aleormenod/triage-ordenes

Ahora conecta a Vercel para tener URL pública con HTTPS.

## OPCIÓN 1: Import Directo (MÁS RÁPIDO - RECOMENDADO)

1. **Abre esta URL:**
   ```
   https://vercel.com/import/project?repo=aleormenod/triage-ordenes
   ```

2. **Vercel te pregunta:**
   - "Login with GitHub" → Acepta
   - "Select repository" → Debería estar preseleccionado
   - Click "Import"

3. **Configuración:**
   - Project Name: `triage-ordenes`
   - Root Directory: `.` (dejar como está)
   - Environment Variables: Ninguno (skip)
   - Click "Deploy"

4. **Espera 30-60 segundos**

5. **Tu URL pública:**
   ```
   https://triage-ordenes.vercel.app
   ```

---

## OPCIÓN 2: Manual en vercel.com

1. Ve a https://vercel.com
2. Crea cuenta / Login
3. Click "New Project"
4. "Import Git Repository" → Selecciona `triage-ordenes`
5. Click "Deploy"
6. Listo ✅
